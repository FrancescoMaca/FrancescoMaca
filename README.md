### Crypto
##### **Language:** C#

This is my best project with a purpose so far. It encodes one or multiple file with a passowrd of your choice. It uses AES with a 256bit key, so in simple terms, it's **very secure**. Everything started with my curiosity about encryption/decryption standards like AES and RSA. So one day I started making a small program that encoded words or sentences. So I started understanding how it was working and when I figured it out I just started working to a more sophisticated way of using it. 
I really like this one because of the graphics I've made and the small details I've added such as opening directory when you click on the window's title. It's not 100% bug-free but the major ones have been fixed in previous version of the program.
